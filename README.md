# semanaomnistack11
Aplicação be the hero, utiliza NodeJS/react/react-native como solução para web e mobile (curso ministrado pela RocketSeat)

Resumo da aplicação: Um projeto onde ONGs podem cadastrar seus casos para receber colaboração da comunidade. 
Os usuarios por sua vez podem acessar os casos, ler seus detalhes e decidir apoiar a ONG com uma quantinha em dinheiro para solucionar
algum caso.

Pasta Back End: Uma API Rest desenvolvida com NodeJS que retorna respostas em JSON para ser consumida pelas outras aplicações.

Pasta Front End: Aplicação Web desenvolvida com React, consome a API que se encontra no back end e permite que as ONGs se cadastrem e 
insiram seus casos para pedir apoio.

Pasta Mobile: Aplicação mobile desenvolvida com react-native onde os usuarios tem acesso aos casos e ao contato das ONGs para ajuda-las e
ser o herói do caso.
